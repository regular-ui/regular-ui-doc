# Regular UI Documentation Builder

[Regular UI Website](http://regular-ui.github.io) is built by this repo.

## Commands

- `gulp`：监听md文件，实时更新文档。
- `gulp watch`：同上
- `gulp doc`：先清理`./doc`目录，然后重新生成文档。
- `gulp page`：先运行`gulp doc`重新生成文档，然后复制到`regular-ui.github.io`目录下。
